## 👀 앱 소개
ARDish APP은 증강현실이라는 시각적 요소를 메뉴판에 접목시켜 </br>
청각장애인 또는 사회적 약자를 위해 식당 음식에 대한 이해도와 접근성, 편의성을 제공합니다.</br>
음식점 메뉴판의 부족한 정보에 대해 점원을 통해 설명을 들을 수 없는 경우를 이 앱을 통해 도움을 줄 수 있도록 개발 목표를 추구했습니다.
</br>

## 🔧 앱 개발 스택
```
- C#
- UnityEngine (2020.3.12)
- Unity AR Foundation Framework
- Figma UI Design
```
